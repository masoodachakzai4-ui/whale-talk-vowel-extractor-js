Whale Talk Vowel Extractor (JavaScript)

This is a simple JavaScript program that extracts vowels from a given string.
It loops through each character of the input text and checks for vowels.
The vowels (a, e, i, o, u) are stored in a new array.
The result is then joined into a string and converted to uppercase.
It demonstrates the use of loops, arrays, and string manipulation.
This project helps practice basic JavaScript logic and iteration.
